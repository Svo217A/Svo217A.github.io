MIT Classes

6.815

Digital and Computational Photography

In 6.815, I implemented several labs applying computational photography techniques such as blur filters, HDR, and morphing in C. For the final project, I implemented several seam carving functions to intelligently resize images. In this class, I was able to see the results of my code and implement a wide variety of image processing algorithms which were really fun to explore.


6.179

Introduction to C and C++

In 6.179, I refreshed myself on C and C++ concepts. The class enabled me to further my understanding of these languages and get familiar with using command line. I worked on a final project with a fellow student, creating a hangman game. This project can be found on my GitHub account.


6.178

Introduction to Software Engineering in Java

In 6.178, I refreshed myself on Java concepts while learning more about the language in great depth. Every week, I completed a problem set consisting of a lab, such as creating a checkers game.
I was a Teaching Assistant for the course in 2019.


6.172

Performance Engineering of Software Systems

In 6.172, I worked on several projects in teams to better optimize existing software, making them faster and more efficient. I learned techniques in understanding someone else's software in order to restructure it for high performance.

6.170

Software Studio

In 6.170, I gained web application development experience using Javascript (JS) and Vue.js through several assignments. I learned about conceptualization, web design, and software engineering ethics and considerations.


6.046

Design and Analysis of Algorithms

In 6.046, I learned about more advanced algorithms such as Fast Fourier Transform, streaming algorithms, and the Gradient Descent Method among many others.


6.042

Mathematics for Computer Science

6.042 covered discrete math for CS, including proofs, graph theory, and probability. I learned how to analyze algorithms and gained a better intuition to probability problems.

6.036

Introduction to Machine Learning

6.036 introduced me to machine learning approaches, strengths, and weaknesses. Through Python, I was able to design machine learning components and get an in depth understanding on how the algorithms interact with each other and with the given data.


6.033

Computer Systems Engineering

In 6.033, I learned about large-scale systems and the design choices behind the implementations of well known systems. Through a final project, I gained experience designing a system, trading off different considerations such as scalability and simplicity.


6.031

Software Construction

6.031 is a course aimed at learning how to code: making programs safe from bugs, easy to understand, and ready for change. I became familiar with modern programming tools such as Eclipse and Git. I learned how to use interfaces, how to prevent deadlock when using concurrent systems, and how to use functional programming.


6.03

Introduction to Electrical Engineering and Computer Science via Medical Technology

In 6.03, I learned the basics of MATLAB. I gained an understanding of the physics and analysis behind common medical technologies: Glucose sensors, Electrocardiograms (ECG), and Magnetic Resonance Imaging (MRI).


6.009

Fundamentals of Programming

In 6.009, I completed Python programming labs to enhance my problem solving abilities. I learned techniques enabling me to more quickly solve complex problems with code.


6.006

Introduction to Algorithms

In 6.006, I learned common algorithms and data structures for problem solving. I learned dynamic programming and algorithmic designs to solve graph problems.


6.004

Computation Structures

In 6.004, I used assembly and Bluespec to program hardware. I learned about caches and multi-cycle processors as well as the tradeoffs behind hardware and software design.
