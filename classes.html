ASFA Classes

ASFA is where I first learned how to program and code. Coding in a variety of languages from Python to Alice to AppInventor.
In the 10th grade, I took AP CS and learned in great depth Java. In my junior year of high school, I learned to use databases using SQL and SAS. Finally, I learned a bit of C and C++ using a LINUX (UNIX) system in senior year.

6.009 - Fundamentals of Programming

Like all courses at MIT, Fundamentals of Programming is commonly referred to by its course number: 6.009. In this class, I completed programming labs to enhance my problem solving abilities. I learned techniques enabling me to more quickly solve complex problems with code.
This class utilized Python and I finished with a B.

6.03 - Intro to EECS via Medical Technology

In 6.03, I learned the physics and analysis behind common medical technologies: Glucose sensors, Electrocardiograms (ECG), and Magnetic Resonance Imaging (MRI).
I learned the basics of MATLAB and finished strong with an A.

6.178 - Introduction to Software Engineering in Java

In 6.178, I refreshed myself on some Java concepts while learning more about the language in great depth. Every week, I completed a problem set consisting of a lab, such as creating a checkers game.

6.179 - Introduction to C and C++

In 6.179, I refreshed myself on some C and C++ concepts. The class enabled me to further my understanding of these languages and get familiar with using command line. I worked on a final project with a fellow student, creating a hangman game. This project can be found on my GitHub account.
